## Вариант A — "Swiss Precision" (Швейцарский минимализм)

**Design Movement:** Международный типографический стиль (Swiss Design) с элементами Bauhaus — функциональная эстетика, где каждый элемент имеет чёткое назначение.

**Core Principles:**
1. Типографика как основной визуальный элемент — крупные заголовки, контрастные размеры
2. Жёсткая модульная сетка с асимметричным балансом
3. Монохромная палитра с одним ярким акцентом
4. Информационная плотность без визуального шума

**Color Philosophy:** Глубокий тёмный фон (#0A0A0F) как "холст" для контента. Единственный акцент — электрический индиго (#6366F1) для интерактивных элементов и прогресса. Серые оттенки для иерархии текста. Философия: "тёмная комната, где светится только важное".

**Layout Paradigm:** Двухпанельный layout — левая панель (1/3) для навигации и управления, правая (2/3) для контента/превью. Верхняя полоса с breadcrumbs и статусом. Нет центрированных hero-секций.

**Signature Elements:**
1. Горизонтальные линии-разделители с номерами секций (01, 02, 03)
2. Прогресс-бар как тонкая горизонтальная полоса сверху страницы, меняющая цвет по фазам
3. Моноширинный шрифт для технических данных (статусы, ID, время)

**Interaction Philosophy:** Мгновенный отклик без лишних анимаций. Hover-эффекты — смена цвета текста, не фона. Фокус на информации, не на декорации.

**Animation:** Минимальные — fade-in 200ms для появления контента, slide-left для панелей. Прогресс-индикатор — плавное заполнение. Никаких bounce или spring.

**Typography System:** 
- Заголовки: Space Grotesk (700, 600) — геометрический гротеск
- Тело: Inter (400, 500) — нейтральный и читаемый
- Код/статусы: JetBrains Mono (400) — моноширинный акцент

</idea>
<text>Швейцарский минимализм с тёмным фоном, модульной сеткой и типографикой как главным визуальным элементом. Двухпанельный layout, электрический индиго как единственный акцент.</text>

## Вариант B — "Glassmorphic Studio" (Студия со стеклянными панелями)

**Design Movement:** Glassmorphism + Spatial Design — вдохновлён Apple Vision Pro и macOS Sonoma. Многослойные полупрозрачные поверхности создают ощущение глубины.

**Core Principles:**
1. Полупрозрачные панели с backdrop-blur как основной паттерн
2. Мягкие градиенты фона, создающие атмосферу "рабочего пространства"
3. Округлые формы и плавные переходы
4. Иерархия через прозрачность и размытие

**Color Philosophy:** Светлый фон с мягким градиентом (бледно-лавандовый → тёплый серый). Стеклянные панели с white/80% opacity и blur. Акцент — насыщенный фиолетово-синий (#4F46E5) для CTA и активных состояний. Зелёный (#10B981) для успеха, оранжевый (#F59E0B) для прогресса.

**Layout Paradigm:** Свободная канва с "плавающими" карточками. Главная — большая центральная форма-карточка с боковыми виджетами. Превью — полноэкранный режим с overlay-панелью управления.

**Signature Elements:**
1. Стеклянные карточки с тонкой белой обводкой (1px border white/20%)
2. Анимированный градиентный фон, медленно перетекающий
3. Круговой прогресс-индикатор с glow-эффектом

**Interaction Philosophy:** Мягкие, "физичные" взаимодействия. Карточки слегка приподнимаются при hover (translateY + shadow). Кнопки имеют subtle glow. Всё ощущается "осязаемым".

**Animation:** Spring-based анимации (framer-motion). Карточки появляются с scale 0.95→1.0 + fade. Переходы между страницами — crossfade. Прогресс — пульсирующий glow.

**Typography System:**
- Заголовки: Plus Jakarta Sans (700, 800) — современный и дружелюбный
- Тело: Inter (400, 500) — чистый и нейтральный
- Акценты: Plus Jakarta Sans (600) — для кнопок и лейблов

</idea>
<text>Glassmorphism с полупрозрачными панелями, мягкими градиентами и spring-анимациями. Вдохновлён Apple Vision Pro — многослойные стеклянные поверхности создают глубину.</text>

## Вариант C — "Editorial Canvas" (Редакционный холст)

**Design Movement:** Editorial Design + New Brutalism — вдохновлён журнальной вёрсткой (Bloomberg, Stripe) с элементами нео-брутализма. Крупная типографика, смелые контрасты, информационная насыщенность.

**Core Principles:**
1. Крупная типографика как навигационный инструмент — размер шрифта = важность
2. Контрастные блоки с чёткими границами (не тени, а линии)
3. Асимметричная сетка с "разрывами" для акцентов
4. Цвет используется точечно и функционально

**Color Philosophy:** Чистый белый фон (#FAFAFA) с чёрным текстом для максимального контраста. Один яркий акцент — коралловый (#FF6B6B) для прогресса и CTA. Тёмно-синий (#1E293B) для secondary элементов. Философия: "газетная чистота с одним ярким маркером".

**Layout Paradigm:** Вертикальный поток с горизонтальными "полосами" разной высоты. Форма создания — на всю ширину с крупным textarea. Список презентаций — таблица с hover-строками. Превью — split-view (список слайдов слева, большой превью справа).

**Signature Elements:**
1. Толстые чёрные линии-разделители (2-4px) между секциями
2. Крупные номера шагов (72px) рядом с описанием прогресса
3. Монохромные иконки-пиктограммы для статусов

**Interaction Philosophy:** Чёткие, "щёлкающие" взаимодействия. Hover — инверсия цвета (чёрный фон, белый текст). Клик — мгновенный отклик без задержки. Фокус на эффективности.

**Animation:** Минимальные, но выразительные. Появление блоков — slide-up 300ms с easing. Прогресс — заполняющаяся полоса с крупным процентом. Переходы — clip-path reveal.

**Typography System:**
- Заголовки: Instrument Serif (400) — элегантный serif для контраста
- Тело: DM Sans (400, 500) — геометрический гротеск
- Акценты/числа: Space Mono (400, 700) — моноширинный для данных

</idea>
<text>Редакционный дизайн с журнальной вёрсткой, крупной типографикой и контрастными блоками. Serif заголовки + моноширинные числа. Коралловый акцент на белом фоне.</text>
