# 1. Введение

**AI Presentation Generator** — это веб-приложение с открытым исходным кодом, предназначенное для автоматического создания высококачественных презентаций на основе текстовых запросов пользователя. Используя мощь больших языковых моделей (LLM), приложение генерирует структуру, текст, дизайн и вёрстку слайдов, позволяя пользователям получать готовые к использованию презентации за считанные минуты.

## Ключевые возможности

- **Генерация из промпта**: Создавайте презентации с нуля, просто описав тему и желаемое содержание.
- **Гибкая настройка дизайна**: Выбирайте из множества профессиональных дизайн-пресетов, чтобы соответствовать вашему бренду или стилю.
- **Редизайн в один клик**: Мгновенно меняйте визуальный стиль существующей презентации, сохраняя при этом её содержание.
- **Экспорт в популярные форматы**: Скачивайте готовую презентацию в форматах **PDF** и **PPTX** для дальнейшего редактирования или демонстрации.
- **Интерактивный интерфейс**: Управляйте процессом генерации через удобный чат-интерфейс и просматривайте артефакты (промежуточные и финальные результаты) в реальном времени.
- **Аутентификация и изоляция данных**: Безопасное хранение проектов для каждого пользователя с полной изоляцией данных.

## Для кого этот проект?

- **Профессионалы и студенты**, которым нужно быстро готовить презентации.
- **Разработчики**, интересующиеся применением LLM в реальных продуктах и full-stack разработкой (Python, FastAPI, React, TypeScript, Docker).
- **Компании**, которые хотят автоматизировать создание внутренних или внешних отчётов и презентаций.

Этот документ проведёт вас через все этапы работы с приложением: от установки и запуска до использования всех его функций.
