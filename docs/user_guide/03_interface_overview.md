# 3. Обзор интерфейса

Интерфейс приложения разделён на три основные части: **Чат**, **Панель артефактов** и **Область просмотра**.

## 1. Чат (Chat)

Это основная область взаимодействия с AI-ассистентом. Здесь вы можете:

- **Отправлять запросы**: Введите тему для новой презентации или команду для редактирования.
- **Получать обновления**: Ассистент информирует о ходе выполнения задачи, показывая текущий шаг (например, «Планирование», «Генерация дизайна», «Создание слайдов»).
- **Просматривать логи**: Все сообщения от AI и системы логируются в чате, что позволяет отслеживать весь процесс.

## 2. Панель артефактов (Artifacts Panel)

Справа от чата находится панель, где отображаются все сгенерированные в ходе работы файлы и результаты. Артефакты — это промежуточные и финальные продукты работы движка.

| Артефакт | Описание |
|:---|:---|
| `s1_context.md` | Анализ первоначального запроса пользователя. |
| `s2_narrative.md` | Структура и текстовое содержание будущей презентации. |
| `s3_design_system.json` | Сгенерированная дизайн-система (цвета, шрифты, стили). |
| `s4_slides.html` | Финальная презентация в формате HTML. |
| `s5_validation_report.md` | Отчёт о качестве и соответствии презентации запросу. |

Для каждого артефакта доступны следующие действия:

- **Просмотр**: Открывает содержимое файла в области просмотра.
- **Скачивание**: Загружает файл на ваш компьютер.
- **Редактирование**: Позволяет вносить изменения в текстовые артефакты (например, в `s2_narrative.md`) и перезапускать генерацию с учётом правок.

## 3. Область просмотра (Viewer)

Центральная и самая большая часть экрана. Здесь отображается содержимое выбранного артефакта. Для HTML-презентаций (`s4_slides.html`) здесь будет показана сама презентация, которую можно пролистывать.

## Панель инструментов (Toolbar)

Над панелью артефактов расположена панель с глобальными действиями:

- **Новый проект**: Начинает новую сессию генерации, очищая текущий контекст.
- **Сменить стиль (Redesign)**: Позволяет применить новый дизайн-пресет к уже сгенерированной презентации.
- **Экспорт**: Предоставляет опции для скачивания финальной презентации в форматах **PDF** и **PPTX**.
